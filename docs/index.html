RFID Tracker Prototype

A Real-Time RFID Dashboard for Tracking Valve Parts Across Workflow Zones

View GitHub Repo View Live App

🔍 What This Is

This is a prototype RFID-based tracking system for managing valve parts through multiple zones (Receiving, Warehouse, Machine Shop, Shipping). It simulates RFID scans, logs history, and presents a live-updating dashboard for real-time tracking.

⚙️ Features

  • 🔄 Live dashboard with polling every 10s
  • 🏷️ Filter parts by zone or job
  • 📍 Color-coded idle time alerts (yellow = 30+ min, red = 2+ hours)
  • 📊 Part history with timestamped zone transitions
  • 📤 CSV export for recordkeeping
  • 🧪 Manual scan simulation interface

🧠 Architecture

  • Frontend: HTML + JavaScript + Chart.js
  • Backend: Python + Flask
  • Database: SQLite (via SQLAlchemy)
  • Deployment: Render.com (backend) + GitHub Pages (docs)

📦 Tech Stack

  • Python 3.11
  • Flask 2+
  • SQLite / SQLAlchemy
  • Chart.js
  • HTML + JavaScript (Vanilla)

📌 How to Use

  1. Scan tags using the simulation form or API
  2. Track part location and technician assignment in real-time
  3. Click "View" to see movement history

🔧 Future Improvements

  • Authentication + user roles
  • Mobile-friendly dashboard
  • Multi-user logging + analytics
